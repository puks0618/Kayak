# Kayak Web Client

User-facing web application built with React and Redux.

## Features

- Search flights, hotels, and car rentals
- Book travel services
- Manage bookings
- User profile management
- AI-powered recommendations

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- Redux Toolkit
- React Router
- Vite
- Axios
