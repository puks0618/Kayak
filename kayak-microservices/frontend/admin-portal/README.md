# Kayak Admin Portal

Administrative dashboard for managing the Kayak platform.

## Features

- Dashboard with analytics
- User management
- Listing management
- Booking oversight
- Reports and charts (Recharts)
- Role-based access control

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- Recharts
- React Router
- Vite
- TanStack Table
